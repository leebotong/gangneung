gulp 셋팅 순서

1. Visual studio code 설치
	-> https://code.visualstudio.com/
	-> Extension : Live server 설치

2. Node.js 설치 - Version 18으로 설치
	-> https://nodejs.org/en

2.  sass 설치(터미널에서)
	-> npm install node-sass@8.0.0

3. 설치 후 터미널에서 아래 명령어 순차적으로 실행
	- npm i
	- npm run dev : css 폴더 아래 content.css 파일 생성
